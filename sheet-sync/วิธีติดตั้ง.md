# วิธีเชื่อม Google Sheet ⇄ ระบบบัญชีจ่าย (ทำครั้งเดียว ~10 นาที)

หลังติดตั้ง: กรอกในชีตแบบเดิม → โผล่ในระบบเองใน 5 นาที | บันทึกในระบบ → เขียนกลับลงชีตทันที

## ขั้นที่ 1 — วางโค้ดในชีต

1. เปิดชีต **"รอทำจ่าย The Good 2026 ทุกๆสิ้นเดือน"**
2. เมนู **ส่วนขยาย (Extensions) → Apps Script**
3. ลบโค้ดเดิมในหน้าต่างที่เปิดขึ้นมาทั้งหมด
4. เปิดไฟล์ `apps-script-code.gs` (อยู่โฟลเดอร์เดียวกับคู่มือนี้) → copy ทั้งหมด → วาง
5. บรรทัด `var TOKEN = 'CHANGE-ME-SECRET';` — เปลี่ยน `CHANGE-ME-SECRET` เป็นรหัสลับที่คิดเอง (ภาษาอังกฤษ+ตัวเลขยาวๆ) **จดไว้ด้วย**
6. กด **บันทึก** (ไอคอนแผ่นดิสก์)

## ขั้นที่ 2 — เปิดเป็น Web App

1. กดปุ่ม **Deploy (ทำให้ใช้งานได้)** มุมขวาบน → **New deployment (การทำให้ใช้งานได้รายการใหม่)**
2. กดไอคอนเฟือง ⚙ ข้าง "Select type" → เลือก **Web app**
3. ตั้งค่า:
   - Execute as (ดำเนินการในฐานะ): **Me (ฉัน)**
   - Who has access (ผู้ที่มีสิทธิ์เข้าถึง): **Anyone (ทุกคน)**
4. กด **Deploy** → กด **Authorize access** → เลือกบัญชี Google ของคุณ → ถ้าขึ้นจอเตือน กด Advanced → Go to ... (unsafe) → Allow
5. จะได้ **Web app URL** ลงท้ายด้วย `/exec` → **copy เก็บไว้**

## ขั้นที่ 3 — บอกระบบให้รู้จักชีต

เปิดไฟล์ `tgm-local\server\.env` ด้วย Notepad แล้วเติม 2 บรรทัด:

```
SHEET_SYNC_URL=วาง Web app URL ที่ได้จากขั้นที่ 2
SHEET_SYNC_TOKEN=รหัสลับที่ตั้งในขั้นที่ 1
```

Save แล้ว **ปิด-เปิดระบบใหม่** (ปิดหน้าต่างดำแล้วดับเบิลคลิก start.bat)

## ขั้นที่ 4 — ทดสอบ

1. เข้าหน้า "บัญชีจ่าย" ในระบบ → กดปุ่ม **⟳ Sync Google Sheet** → ควรขึ้น "ดึงจากชีตแล้ว ... รายการ"
2. ลองพิมพ์รายการใหม่ในชีต → รอ 5 นาที (หรือกดปุ่ม Sync) → รายการโผล่ในระบบ
3. ลองแก้ในระบบแล้วกดบันทึก → เปิดชีตดู → ข้อมูลเปลี่ยนตาม

## กติกาที่ควรรู้

- ระบบจะเติมรหัสในคอลัมน์ **P** ของชีต (คลิกขวาหัวคอลัมน์ → ซ่อน ได้เลย **ห้ามลบค่าข้างใน**)
- ช่องติ๊กถูก ✓ ในชีต = สถานะ "จ่ายแล้ว" ในระบบ (ติ๊กในชีต ระบบเปลี่ยนตาม และกลับกัน)
- คอลัมน์ E (ยอดขาย) ระบบไม่ยุ่ง — ใช้ต่อได้ตามเดิม
- ถ้าแก้รายการเดียวกันพร้อมกันสองฝั่ง "ฝั่งที่แก้ล่าสุดชนะ"
- รายการที่ "ยกเลิก" ในระบบ จะไม่ถูกเขียนลงชีต
- ลบแถวในชีต รายการในระบบยังอยู่ (ต้องลบในระบบเอง — กันลบพลาด)
- แท็บที่ sync คือ **Q2 26** — ถ้าขึ้นแท็บใหม่ (เช่น Q3 26) ให้แก้ชื่อในโค้ด Apps Script บรรทัด `var TAB` แล้ว Deploy → Manage deployments → Edit → Version: New → Deploy
