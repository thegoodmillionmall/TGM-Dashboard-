# คู่มือเปิด-ปิดระบบ TGM Dashboard

## วิธีเปิดระบบ (แบบง่าย — แนะนำ)

1. เปิดโฟลเดอร์ `Documents\TGM\tgm-local`
2. **ดับเบิลคลิก `start.bat`**
3. จะมีหน้าต่างดำเด้งขึ้นมา 2 หน้าต่าง แล้วเบราว์เซอร์เปิด http://localhost:5173 ให้เอง
4. login: username `admin` + รหัสผ่านของคุณ

> หน้าต่างดำ 2 อัน = เครื่องยนต์ของระบบ **ต้องเปิดค้างไว้ตลอดเวลาที่ใช้งาน**
> ย่อหน้าต่างได้ แต่ห้ามกดปิด (X) — ปิดเมื่อไหร่เว็บจะเข้าไม่ได้ทันที

## วิธีเปิดระบบ (แบบมือ — ผ่าน VS Code)

เปิด VS Code → Terminal → พิมพ์:

```
cd C:\Users\UsEr\Documents\TGM\tgm-local\server
npm run dev
```

รอจนขึ้น `TGM Local API รันที่ http://localhost:3001` แล้วกดปุ่ม **+** เปิด terminal ที่สอง:

```
cd C:\Users\UsEr\Documents\TGM\tgm-local\web
npm run dev
```

รอจนขึ้น `Local: http://localhost:5173` → เปิดเบราว์เซอร์ไปที่ **http://localhost:5173**

## วิธีปิดระบบ

ปิดหน้าต่างดำทั้ง 2 อัน (หรือปิด terminal ใน VS Code) — แค่นั้น ข้อมูลไม่หายเพราะเก็บใน Supabase บนคลาวด์ทั้งหมด

## อาการที่พบบ่อย + วิธีแก้

| อาการ | สาเหตุ | วิธีแก้ |
|---|---|---|
| เว็บขึ้น "ไม่สามารถเข้าถึงเว็บไซต์นี้" / ERR_CONNECTION_REFUSED | ระบบยังไม่ได้เปิด หรือหน้าต่างดำถูกปิดไป | ดับเบิลคลิก `start.bat` ใหม่ |
| login ไม่ผ่านทั้งที่รหัสถูก | หน้าต่าง API (3001) ไม่ได้เปิด หรือ `.env` ค่าไม่ครบ | เช็คว่าหน้าต่าง "TGM API" ยังเปิดอยู่ |
| ตัวเลขเป็น 0 หมด | ช่วงวันที่ที่เลือกไม่มีข้อมูล | เปลี่ยนช่วงวันที่ให้ครอบคลุมข้อมูล แล้วกด "แสดงข้อมูล" |
| อัปโหลดไฟล์ inbox แล้วไม่เข้า | ตั้งชื่อไฟล์ไม่ตรงรูปแบบ | ดูไฟล์ `.สาเหตุ.txt` ในโฟลเดอร์ `inbox\error` และ `README.txt` ใน inbox |
| แก้ข้อมูลแล้วหน้าเว็บไม่เปลี่ยน | ยังไม่ได้กดปุ่มบันทึก / ยังไม่ refresh | กดบันทึก แล้วกด F5 |

## งานประจำที่ต้องทำ

- **ทุกเช้า/ทุกสัปดาห์:** ดาวน์โหลดไฟล์ยอดขายจาก Seller Center → เปลี่ยนชื่อตามรูปแบบ (เช่น `TiktokOrder_xxx.csv`) → ลากไปวางในโฟลเดอร์ `tgm-local\inbox` → ระบบดูดเข้าเองใน 10 นาที
- **ทุกสิ้นเดือน:** อัปโหลดไฟล์ Settlement + Affiliate
- **บัญชีจ่าย:** กรอกที่หน้า "บัญชีจ่าย" → กดปุ่ม "สรุปยอดจ่าย" เพื่อ copy ข้อความส่งไลน์ (ดึงเฉพาะรายการที่ยังไม่จ่ายของวันที่เลือก)

## ที่อยู่สำคัญ

- หน้าเว็บระบบ: http://localhost:5173
- โฟลเดอร์วางไฟล์อัตโนมัติ: `Documents\TGM\tgm-local\inbox`
- ค่าเชื่อมต่อระบบ (ห้ามแชร์ให้ใคร): `Documents\TGM\tgm-local\server\.env`
- คู่มือติดตั้งฉบับเต็ม: `Documents\TGM\tgm-local\README.md`
